# 📚 README – Atividade de Vetores em JavaScript

## 📝 Descrição

Esta atividade teve como objetivo desenvolver programas em JavaScript utilizando vetores para armazenamento e manipulação de dados. Os exercícios propostos exploram conceitos de entrada de dados, estruturas de repetição, condicionais e percursos em vetores, sem a utilização de funções específicas para arrays.

Além da implementação dos algoritmos, foi realizado o versionamento dos arquivos utilizando Git e a publicação do projeto em um repositório remoto no GitHub.

---

## 🎯 Objetivos

* ✅ Desenvolver programas em JavaScript (.js).
* ✅ Aplicar conceitos de vetores para armazenar informações.
* ✅ Utilizar estruturas de repetição e condicionais.
* ✅ Realizar cálculos e buscas dentro de vetores.
* ✅ Testar os programas para garantir seu funcionamento correto.
* ✅ Utilizar Git e GitHub para versionamento e compartilhamento do projeto.

---

## 🚀 Atividades Desenvolvidas

### 👥 1. Média de Idades

Foi desenvolvido um programa que realiza a leitura da idade de oito pessoas e armazena os valores em um vetor. Ao final, o algoritmo calcula e exibe a média das idades cadastradas.

### 🔎 2. Pessoas com Mais de 25 Anos

O programa lê a idade de sete pessoas e armazena os dados em um vetor. Após o preenchimento, são exibidas as posições onde estão armazenadas as idades superiores a 25 anos.

### 🔢 3. Múltiplos de 10 em um Vetor

Foi criado um algoritmo que realiza a leitura de quinze números inteiros e os armazena em um vetor. Ao final, o programa exibe todos os valores cadastrados e identifica as posições que contêm números múltiplos de 10.
